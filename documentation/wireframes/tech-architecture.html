TCG World's architecture is built for scalability, flexibility, and creator-friendliness. Our platform combines Unity's 3D capabilities with powerful AI systems to transform natural language into playable card games.

What makes our platform truly unique is the AI-powered rules engine that allows creators to define their game mechanics in plain English rather than code.

  return attacker.state.energy >= 1

  attacker.state.energy = attacker.state.energy - 1
  engine:dealDamage(attacker, target, attacker.state.attack)
end

Creator Prompt

"When this card is played, destroy a target enemy creature."

LLM-Generated Script 
definition:addHook("onPlay", function(player, card)
  engine:promptChoose(player, {type="enemy", zone="battlefield"}, function(target)
    engine:destroy(target)
  end)
end)
Execution
  • This script gets attached to the card definition object.
  • The engine loads all hooks at runtime and dispatches triggers.
